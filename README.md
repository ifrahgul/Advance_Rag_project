
# Advanced RAG Streamlit App

An AI-powered Retrieval-Augmented Generation (RAG) application built with Streamlit. This project enables intelligent document querying, sentiment analysis, and context-aware responses using modern NLP techniques.

---

## Features

- Retrieval-Augmented Generation (RAG)
- Interactive Streamlit Interface
- Context-Aware AI Responses
- Sentiment Analysis using TextBlob
- Real-Time User Interaction
- Easy Deployment and Setup

---

## Technologies Used

- Python
- Streamlit
- NLP
- TextBlob
- Machine Learning Concepts
- RAG Architecture

## Setup Instructions

### 1. Create Virtual Environment

```bash
python -m venv venv
```

### 2. Activate Virtual Environment

#### Windows
```bash
.\venv\Scripts\activate
```

#### Linux/Mac
```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Download TextBlob Corpora

```bash
python -m textblob.download_corpora
```

### 5. Run the Application

```bash
streamlit run app.py
```
